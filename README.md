进入目录
```bash
cd moon-web-start
```

安装依赖
```bash
npm install -g pnpm
pnpm install
```

启动项目
```bash
npm run dev
```

# docker容器

```shell
docker pull luode0320/web-start:1.0
```

启动:

```shell
docker run -d -p 2000:2000 luode0320/web-start:1.0
```
